h1. First steps

As pentesters we are always running tools to gather information, generating the knowledge base of the current project, and most often we need to get a grip to that information, as a csv, txt or any file format.

h2. Faraday supports

Actually Faraday supports CSV Exporting from it's UI, just right click in the workspace and export to a csv file.

The current output format is in the form:

found date | severity level | service parent | target | name | vulnerability description | data field | references
